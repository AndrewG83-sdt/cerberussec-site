# Learning Journey: Epicode e Laboratori

Questa pagina serve a contestualizzare le repository nate durante il mio percorso di studi con Epicode.

## Perche le mantengo pubbliche

Non le considero "rumore". Le mantengo online perche raccontano la traiettoria completa:
- primi esercizi
- laboratori guidati
- repository datate per pratica quotidiana
- passaggio progressivo da esercitazioni generiche a casi cybersecurity piu strutturati

## Come interpretare i nomi

- `epicodepdf`, `pratica1esercizio`, `eserciziopratico2`, `laboratorio...`: materiali ed esercitazioni del percorso formativo
- `ex...`: repository di esercizi puntuali o laboratori datati
- `report...` / `REPORT...`: elaborati tecnici o consegne strutturate
- repository con date nel nome: snapshot del lavoro svolto in specifiche giornate o moduli

## Cosa guardare prima

Se vuoi valutare il mio livello attuale, parti dai repository in evidenza nel [README principale](./README.md).

Se vuoi capire l'evoluzione del percorso, queste repository di studio mostrano:
- continuita
- apprendimento progressivo
- esposizione a strumenti e contesti diversi
- passaggio da esercizi guidati a progetti e report piu autonomi

## Stato delle repository di studio

- contenuti originali preservati
- nessuna cancellazione del materiale storico
- repository mantenute come documentazione del percorso, non come portfolio principale

## Nota etica

Anche quando compaiono strumenti o terminologia sensibile, il contesto e didattico, di laboratorio o di analisi difensiva autorizzata.
